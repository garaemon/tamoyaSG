How to Order

Self Service style is a style that allows customers to arrange the Udon to their liking

Choose the type of Udon, and the amount you would like. Freshly made Udon will be served in a bowl.

If you would like hotserved, please lightly boil the noodles. If you would like cold noodles, please move on to step 3. This is true Sanuki style.

Toppings, Rice dishes, and even sushi can be selected here.

For Kake Udon, pour the Kake broth over the noodles. For Bukkake and Kamatama Udons, pour the Tsuke Broth over the noodles. For Kamaage and Zaru Udons, add the Tsuke broth into a separate bowl to use as dipping sauce.

All condiments are free, so payments will be done here.

Lastly, add green onions, ginger, grated radish, seaweed, and tempura flakes to your liking.

After you finish eating, please return the trays, bowls, and plates to the Return Box.
